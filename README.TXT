This project consists of a panorama stich code which detects sift features from two adjacent images first and then matches 
their features and finally stiches them together

Panorama.py consists of complete implementation of panorama code 

Sift.py consists of sift module's implementation which is used in Panorama.py

Run Panorama.py to stich two images together on the basis of SIFT features
